Aksi Kami

Quit Coal

Aktivis Greenpeace Indonesia membentangkan banner dengan pesan "Quit Coal", saat mereka memblokade crane yang akan memuat batu bara di PLTU batu bara Cirebon, Jawa Barat. Aksi ini menjadi bagian dari gerakan aksi global bertajuk Break Free yang berlangsung selama 4-15 Mei 2017 dan mendesak pemerintah untuk tidak lagi mengeksploitasi batu bara, minyak bumi, dan gas alam. © Ardiles Rante

Tim Cegah Api di Kalimantan Barat

Tim cegah Api Greenpeace memadamkan api di lahan gambut yang terbakar di wilayah Desa Punggur Kecil, Kec. Sungai Kakap, Kab. Kubu Raya, Pontianak, Kalimantan Barat. Tim Cegah Api ditugaskan di area lahan gambut untuk mencegah api menjalar dan melakukan investigasi. © Rendra Hernawan

Kebakaran Hutan Papua

Greenpeace menginvestigasi dan mendokumentasikan kebakaran pada lahan gambut di dekat PT Tunggal Mitra Plantation - sebuah perusahaan kelapa sawit milik kelompok Sime Darby, anggota Roundtable on Sustainable Palmoil (RSPO). Menurut grafik resmi terbaru, daerah di Desa Pujud, Kabupaten Rokan Hilir, Riau, Sumatra dicakup oleh moratorium pemerintah Indonesia tentang penerbitan izin baru di hutan primer dan lahan gambut. © Ulet Ifansasti
